# 会話ネタジェネレーター

日本語レッスン中に会話トピックをランダムで表示するアプリ。

## 使い方

1. レベルを選ぶ（初級・中級・上級）
2. 「次のネタ🎲」を押してトピックを表示
3. 「英語訳を見る」で英語訳を確認
4. 「ヒントを見る」で単語・文型を確認
5. ☆を押してお気に入り保存
6. タイマーで会話時間を計る（3分・5分）

## 機能

- レベル別ネタ表示（初級・中級・上級 各20問）
- フリガナ付き日本語表示
- 英語訳・ヒント（単語・文型）アコーディオン表示
- お気に入り保存（ブラウザを閉じても保持）
- セッション履歴（今日使ったネタ一覧）
- タイマー（3分・5分）
- 重複防止（同じネタが連続しない）

## 技術

HTML + CSS + JavaScript（単一ファイル・デプロイ不要）
